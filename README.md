# Todo App Backend

A simple backend for a Todo applicataion using **Node.js**, **Express.js** , **Mongodb**

## Live Site -- https://todo-backend-flame-ten.vercel.app/

---
## Features
- Create a new todo
- Get all todos
- Update a todo
- Delete a todo
- Postman For API testing

---
## Tech using
- Node.js
- Express.js
- Mongodb
- dotenv
- cors
